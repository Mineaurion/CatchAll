BUKKIT :

permission : mineaurion.reload
/MineaurionReload -> Reload le plugin

permission : mineaurion.lastlogin
/lastloggin (nombreDeMois) -> Génère un fichier avec la liste des afks depuis 1 mois

permission : mineaurion.lanceTomate
/lancetomate <player> -> lance une boule de neige sur le joueurs cooldown 60 secondes


permission : mineaurion.webhook
/webhook <message> - > envoie le message sur le discord

permission : mineaurion.maintenance
/maintenance -> passe le serveur en mode maintenance

permission : Maintenance.bypass
Permet de ce connecter lors de la maintenance


donateur : mineaurion.donateurCommand
/donateur -> fait la meme chose que la maintenance mais seulement pour les donateurs avec la permission mineaurion.donateur

+ commande du plugin ADDONS
